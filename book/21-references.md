### Dog-Safe Foods Quick Reference

| ‚úÖ Safe Foods | Notes |
|--------------|-------|
| **Lean meats** | Chicken, turkey, beef (cooked, unseasoned, no bones) |
| **Fish** | Salmon, whitefish (cooked, deboned, no seasoning) |
| **Eggs** | Cooked eggs are excellent protein |
| **Peanut butter** | Must be xylitol-free, natural preferred |
| **Pumpkin** | Plain canned (not pie filling), great for digestion |
| **Sweet potato** | Cooked, plain, high in fiber and vitamins |
| **Carrots** | Raw or cooked, good for teeth |
| **Green beans** | Plain, fresh or frozen |
| **Apples** | Remove seeds and core |
| **Blueberries** | Antioxidant-rich, great training treats |
| **Watermelon** | Seedless, no rind |
| **Bananas** | In moderation (high sugar) |
| **Oatmeal** | Cooked, plain, good fiber source |
| **Rice** | White or brown, plain |
| **Cheese** | Low-fat varieties, in moderation |
