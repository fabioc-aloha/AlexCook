# Contributing to The Alex Cookbook

## Adding a New Recipe

1. Use the template in `templates/recipe-template.md`
2. Place your recipe in the appropriate chapter folder
3. Update the chapter's README with a link to your recipe
4. Add appropriate tags and difficulty level

## Recipe Guidelines

- Be specific with measurements
- Include prep and cook times
- Add helpful tips in Chef's Notes
- Include photos when possible (place in `assets/images/`)

## Formatting Standards

- Use Markdown for all recipes
- Follow the established template structure
- Keep ingredient lists organized (group by type if helpful)
- Number your steps clearly
